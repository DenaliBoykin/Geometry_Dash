# Geometry Dash Style Game

A simple Geometry Dash–inspired browser game built with Phaser 3 and Vite.

## Features

- One-button jump gameplay
- Random spikes and block obstacles
- Score counter
- Restart on death
- No external art assets required

## Controls

- **Space**
- **Left Click**
- **Tap**

## Getting started

1. Install Node.js
2. Clone or download this repo
3. Run:

```bash
npm install
npm run dev
